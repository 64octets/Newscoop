EXAMPLE TEMPLATES OF TRANSITIONS ONLINE

The following templates were created by Aleksandar Brajanoski <braja@ijt.cz>, webmaster of Transitions Online (http://www.tol.cz) who are running their application on CAMPFIRE and helped developing the software. The templates supplied here as examples are meant to illustrate the functionality of the CAMPSITE parser.

Templates were frozen on the 15th of March 2001.
Developed for CAMPSITE V1.0 (http://www.campware.org)

FILE NAME DEPENDENCY DESCRIPTION
article.tpl stand alone This template is the stand alone template for articles. It builds the navigation and includes the banner ad. Depending on what section you are in, it calls other templates to display the articles (such as letter.tpl or job.tpl).
do_login.tpl stand alone Stand alone template for readers to log in.
emlist.tpl section.tpl Used on section pages. When a new issue is published but there was not enough time to include new material in one section, this template will display the message that there is no new material and generate a list of the articles of the last issue.
home.tpl stand alone Displaying the home page as a stand alone template. It checks if there is some article in the first section with the article type 'homeintro' and displays just the newest one of this type. The layout of this article is done via this template (changed regularly and manually to avoid a too static appearance of the home page).
job.tpl article.tpl Used by article.tpl to display a single job article.
jobs.tpl section.tpl Template which the section template uses to display company, position and date on the job section home page.
jobside.tpl article.tpl The template used by article.tpl for the display of the left, grey column to list other job ads.
letter.tpl article.tpl Template which displays static sections on article level. Originally developed for the 'Letters to the editor' section and now also used for other static sections, such as 'about us'.
login.tpl stand alone Stand alone template, called after you logged in via the do_login.tpl. It will check via the parser if the login was okay. Returns either 'okay' or the subscription (which is a static page, not using a template).
mbs.tpl article.tpl 'Media Bites' template. Used by article template to display in the media section the articles with the type 'mbites'.
not_allowed.tpl article.tpl If the user is not allowed to access a protected article, the article template will pull this template to notify the user to either log in or subscribe.
ourtake.tpl article.tpl Used by article.tpl to display articles with the type 'ourtake' in the section 'Our Take'.
search.tpl stand alone Used to display the serach results. Stand alone template.
secintro.tpl section.tpl Used by section template. When there is an introduction to the section (such as in 'Focus') then this is used to display the intro text. Uses article with the type 'sectionintro'.
section.tpl stand alone Stand alone. Builds the navigation and banner and pulls other templates.
tol.tpl article.tpl Used by article template to display the article which the type 'tol'. Such article types are used in many sections.
tolprint.tpl stand alone Stand alone template to display the article in a printer friendly version, just including title, text, image and copyright notice.
tolsec.tpl section.tpl Used by section template to display a list of articles on the section pages which are of the type 'tol'. This will display title, author, deck and date in the section listing. The title will be linked to the article itself.
wirarticle.tpl article.tpl Used by the article template, to display article with the type 'wir' (Week in Review). See also tol.tpl, which is similar for different article type.
wirsection.tpl section.tpl Used by the section template to display the listing of articles with the type 'wir' in one section. It displays the subheaders of the article and the name of the author.
